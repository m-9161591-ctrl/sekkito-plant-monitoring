◎ SEKKITO CAMERA DEBUG

Standalone diagnostic tool — open this alongside your main dashboard to trace the full image transfer pipeline.

⚠ Common cause of missing images: Browser MQTT.js has a default max packet size of 1 MB. A VGA JPEG can be ~30–60 KB raw, but Base64 encoded = ~80 KB. Each chunk is 512 bytes so ~160 chunks are sent. If any chunk is dropped silently, the image buffer is corrupted. This tool measures exactly what's happening.
